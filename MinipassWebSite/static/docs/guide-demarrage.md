# Guide de demarrage rapide

Bienvenue sur minipass! Ce guide vous aidera a configurer votre compte et a creer votre premiere activite en quelques minutes.

## 1. Connexion a votre compte

Apres votre inscription, vous recevrez un courriel avec vos identifiants de connexion:

1. Rendez-vous sur votre sous-domaine: `votreorganisation.minipass.me`
2. Entrez votre adresse courriel et mot de passe
3. Cliquez sur **Connexion**

## 2. Tableau de bord

Une fois connecte, vous arrivez sur votre tableau de bord. Vous y trouverez:

- **Activites**: Gerez vos activites et evenements
- **Participants**: Consultez la liste de vos inscrits
- **Paiements**: Suivez les transactions
- **Passes**: Generez et validez les passes numeriques

## 3. Creer votre premiere activite

1. Cliquez sur **Activites** dans le menu
2. Cliquez sur le bouton **Nouvelle activite**
3. Remplissez les informations:
   - Nom de l'activite
   - Description
   - Date et heure
   - Tarif
   - Nombre de places disponibles
4. Cliquez sur **Enregistrer**

## 4. Gerer les inscriptions

Quand un participant s'inscrit:

1. Vous recevez une notification par courriel
2. Le participant apparait dans votre liste
3. Vous pouvez valider son paiement
4. Un passe numerique est genere automatiquement

## 5. Scanner les passes

Pour valider les presences:

1. Ouvrez l'application minipass sur votre telephone
2. Cliquez sur **Scanner**
3. Pointez la camera vers le code QR du participant
4. La validation est instantanee!
