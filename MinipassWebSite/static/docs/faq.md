# Questions frequentes (FAQ)

Trouvez rapidement les reponses aux questions les plus courantes sur minipass.

---

## Compte et abonnement

### Comment puis-je m'inscrire?

Rendez-vous sur [minipass.me](https://minipass.me) et cliquez sur **S'abonner**. Choisissez votre forfait et completez le paiement. Votre compte sera cree automatiquement.

### Quels modes de paiement acceptez-vous?

Nous acceptons les paiements par carte de credit (Visa, Mastercard, American Express) via notre plateforme securisee Stripe.

### Puis-je changer de forfait?

Oui! Contactez-nous a support@minipass.me pour modifier votre forfait a tout moment.

---

## Activites et evenements

### Combien d'activites puis-je creer?

Cela depend de votre forfait. Consultez notre page de tarification pour voir les limites de chaque plan.

### Puis-je modifier une activite apres l'avoir creee?

Oui, vous pouvez modifier tous les details d'une activite a tout moment depuis votre tableau de bord.

### Comment supprimer une activite?

Rendez-vous dans la section **Activites**, selectionnez l'activite concernee et cliquez sur **Supprimer**. Attention: cette action est irreversible.

---

## Paiements et transactions

### Comment recevoir les paiements des participants?

Les participants peuvent payer par virement Interac. Vous recevrez une notification lorsqu'un paiement est detecte et il sera automatiquement associe au bon participant.

### Quand est-ce que je recois les fonds?

Les fonds sont deposes directement dans votre compte bancaire lors du virement Interac.

### Comment gerer les remboursements?

Les remboursements se font directement entre vous et le participant. Minipass ne gere pas les transactions financieres directes.

---

## Passes numeriques

### Comment fonctionnent les codes QR?

Chaque participant inscrit recoit un code QR unique. Ce code peut etre scanne pour valider sa presence ou son acces a une activite.

### Puis-je scanner les passes sans connexion internet?

Non, une connexion internet est necessaire pour valider les passes en temps reel et mettre a jour le statut.

### Le code QR est-il securise?

Oui, chaque code QR est unique et crypte. Il ne peut etre utilise qu'une seule fois par session d'activite.

## Support technique

### L'application ne fonctionne pas. Que faire?

1. Rafraichissez la page (F5 ou Ctrl+R)
2. Videz le cache de votre navigateur
3. Essayez un autre navigateur
4. Contactez-nous si le probleme persiste

### Comment contacter le support?

- **Courriel**: support@minipass.me
- **Discord**: [Rejoindre notre serveur](https://discord.gg/mfzBs8pT26)
- **Delai de reponse**: Generalement sous 24 heures

### Ou puis-je suggerer des ameliorations?

Nous adorons recevoir vos commentaires! Envoyez vos suggestions a support@minipass.me ou rejoignez notre Discord pour discuter directement avec l'equipe.

---

## Securite et confidentialite

### Mes donnees sont-elles en securite?

Oui, nous utilisons le chiffrement SSL pour toutes les communications. Vos donnees sont stockees de maniere securisee et ne sont jamais partagees avec des tiers.

### Puis-je exporter mes donnees?

Oui, contactez-nous pour obtenir une copie de vos donnees au format CSV.

### Comment supprimer mon compte?

Envoyez une demande a support@minipass.me. Nous traiterons votre demande sous 48 heures.
