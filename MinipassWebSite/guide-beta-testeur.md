Bienvenue et merci de participer au programme bêta de **Minipass** !

Ce guide vous explique comment activer votre accès en 5 étapes simples à l'aide du code promo que vous avez reçu.

---

## Étape 1 — Rendez-vous sur minipass.me

Sur la page d'accueil, cliquez sur le bouton **« Voir les forfaits »**.

![Étape 1 — Page d'accueil Minipass](STEP1_IMAGE)

---

## Étape 2 — Choisissez votre forfait

Vous verrez les trois forfaits disponibles : Solo, Club et Organisation. Cliquez sur **« M'abonner »** sous le forfait qui correspond à votre code promo.

![Étape 2 — Choisir un forfait](STEP2_IMAGE)

---

## Étape 3 — Remplissez le formulaire

Une fenêtre s'ouvre. Remplissez les trois champs :

- **Nom de votre application** (ex. : `monclub`) → votre URL sera `monclub.minipass.me`
- **Nom de votre organisation** → apparaîtra dans votre application
- **Adresse courriel** → identifiant administrateur et adresse de réception pour tous les courriels Minipass (confirmations, paiements, notifications)

Ensuite, cliquez sur le lien **« Vous avez un code promotionnel ? »** en bas du formulaire.

![Étape 3 — Formulaire d'abonnement](STEP3_IMAGE)

---

## Étape 4 — Entrez votre code promo et cliquez sur « Appliquer »

Saisissez le code promo reçu dans le champ, puis cliquez sur **« Appliquer »**. Une confirmation verte indique que le code est valide. Cliquez ensuite sur **« Procéder au paiement »**.

![Étape 4 — Entrer le code promo](STEP4_IMAGE)

---

## Étape 5 — Attendez votre courriel de bienvenue

Minipass déploie automatiquement votre application. Vous verrez une page indiquant que le déploiement est en cours.

![Étape 5 — Déploiement en cours](STEP5_IMAGE)

**Dans les 10 minutes**, vous recevrez un courriel contenant :

- L'URL de votre application (`votrenom.minipass.me`)
- Votre identifiant et mot de passe administrateur
- Un lien vers la **vidéo de démarrage** pour configurer votre première activité

---

## Des questions ?

N'hésitez pas à nous écrire directement. Votre retour en tant que bêta-testeur est précieux — c'est grâce à vous que Minipass s'améliore.

Merci de faire partie de l'aventure !
