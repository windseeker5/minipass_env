Bonjour!

Voici ce qui a changé depuis le 18 mars.

## Ce qui a été corrigé

- **Changement de plan** — Il était impossible de passer d'un plan mensuel à un plan annuel (ou inverse). C'est maintenant réglé — le changement se fait correctement et la facturation Stripe suit.

## Ce qui a été ajouté

- **Mot de passe oublié** — Les utilisateurs peuvent maintenant demander une réinitialisation de mot de passe directement depuis l'écran de connexion. Un lien valide une heure leur est envoyé par courriel.

- **Associer un paiement Interac à un passeport existant** — Nouvelle fonctionnalité développée pour le cas où un parent paie par Interac pour son enfant. Quand le nom du payeur ne correspond pas automatiquement au passeport, il est maintenant possible de faire le lien manuellement directement depuis la liste des paiements non appariés.

- **Numéro de passeport dans l'export CSV** — Le rapport financier inclut maintenant le numéro de passeport dans le fichier exporté, ce qui facilite le suivi par participant.
